isOptional

Validator: Shows that a value is optional, and only runs validators on it if actually present

isOptional(value: Any, callback: Function)
Parameters
value (Any) The value to check
callback (Function) Provides a validator as the only argument, and should return the validator after adding any needed validations. These validations will only run if the given value is present.

test ((Boolean | Function | Promise)) The test to run for this validator. Can be either a pre-calculated boolean, a Promise function, or a Promise object
msg (String) The error message to show if the test fails
code (Integer) The HTTP status code to return if the test fails
Returns
ValidatorObject: A ValidatorObject is a Promise whose error handler takes a ValidatorError object
